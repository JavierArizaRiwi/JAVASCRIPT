# Fundamentos de JavaScript

Esta carpeta contiene ejemplos básicos que te ayudan a entender la lógica principal de JavaScript.

## Ejemplos incluidos

- `index.js`: ejemplo simple de calculadora usando `readline` en Node.js.
- `ejemplo_if.js`: estructuras condicionales `if`, `else if` y `else`.
- `ejemplo_switch.js`: uso de `switch` con `case` y `default`.
- `ejemplo_for.js`: bucle `for`, `break` y `continue`.
- `ejemplo_while.js`: bucle `while`.
- `ejemplo_foreach.js`: recorrido de arreglos con `forEach`.
- `ejemplo_funciones.js`: funciones tradicionales, anónimas y flecha.
- `ejemplo_objetos.js`: objetos, propiedades y métodos.
- `ejemplo_arreglos.js`: uso de `map`, `filter`, `reduce` e `includes`.
- `ejemplo_metodos.js`: métodos comunes de arreglos y cadenas.
- `ejemplo_tipos_parseos.js`: tipos de datos y conversión entre valores.
- `ejemplo_trycatch.js`: manejo de errores con `try`, `catch` y `finally`.
- `ejemplo_async_await.js`: promesas y `async/await`.
- `ejemplo_debug.js`: ejemplo para practicar debugging con breakpoints y depurador paso a paso.

## Cómo ejecutar

1. Abre la terminal en la carpeta `fundamentos`.
2. Ejecuta el ejemplo que quieras con `node`.

```bash
cd fundamentos
node ejemplo_if.js
node ejemplo_for.js
node ejemplo_arreglos.js
```

Cambia el nombre del archivo por el ejemplo que quieras ejecutar.

## Recomendación

1. Empieza por `index.js` y `ejemplo_if.js`.
2. Avanza con bucles y condicionales.
3. Luego practica funciones, objetos y arreglos.
4. Termina con manejo de errores, async y clases.
