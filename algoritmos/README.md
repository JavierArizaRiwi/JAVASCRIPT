# Carpeta algoritmos

Esta carpeta contiene ejemplos de algoritmos básicos en JavaScript.

## Archivos

- `ejemplo_ordenamiento.js`: muestra bubble sort e insertion sort.
- `ejemplo_busqueda.js`: muestra búsqueda lineal y búsqueda binaria.

## Cómo ejecutar

Usa la terminal dentro de `algoritmos`:

```bash
node ejemplo_ordenamiento.js
node ejemplo_busqueda.js
```

Puedes seguir agregando ejemplos de:

- recursión
- programación dinámica
- gráficas y árboles
