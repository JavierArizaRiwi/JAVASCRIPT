# Manipulación del DOM en JavaScript

Esta carpeta muestra cómo trabajar con el DOM en el navegador usando JavaScript.

## Archivos

- `index.html`: página que carga el ejemplo de tareas.
- `ejemplo_dom.js`: código que crea, guarda y muestra tareas.

## Qué aprender

- Seleccionar elementos del DOM con `getElementById`.
- Crear y modificar elementos con `createElement`.
- Usar eventos como `submit` y `click`.
- Guardar datos en `localStorage` y `sessionStorage`.
- Cambiar clases para alternar temas.

## Cómo usar

Abre `dom/index.html` en tu navegador. Al cargar la página podrás:

- agregar tareas
- eliminar tareas
- guardar y cargar tareas desde el navegador
- cambiar el tema visual

### Pasos para usar el ejemplo

1. Abre `dom/index.html` en el navegador.
2. Escribe una tarea y presiona "Agregar tarea".
3. Guarda las tareas con "Guardar en localStorage" o "Guardar en sessionStorage".
4. Recarga la página y usa "Cargar desde localStorage" para ver cómo persisten los datos.

## Consejo

Lee los comentarios en `ejemplo_dom.js` para entender cada parte del ejemplo. Si quieres practicar, cambia el texto de las tareas o agrega nuevas funcionalidades.
